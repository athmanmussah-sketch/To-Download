**DarkX Video Downloader** ni tool ya kupakua video na music kutoka YouTube kwa kutumia Termux. Imeboreshwa kwa ajili ya matumizi rahisi na ina features nyingi za kipekee.

## ✨ Features
- ✅ Download video za YouTube kwa ubora tofauti (4K, 1080p, 720p, 480p)
- ✅ Download audio MP3 tu
- ✅ Support playlist nzima
- ✅ Interface ya rangi na user-friendly
- ✅ Auto-detect quality
- ✅ Embed thumbnail na metadata
- ✅ Multiple format support (MP4, MKV, MP3)
- ✅ Show recent downloads with file sizes
